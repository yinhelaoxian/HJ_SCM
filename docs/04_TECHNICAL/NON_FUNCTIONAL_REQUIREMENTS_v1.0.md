# 非功能性需求规范

**版本**: v1.0  
**状态**: Draft  
**更新日期**: 2026-02-14  
**适用范围**: HJ_SCM 全系统  

---

## 1. 性能要求

| 场景 | 指标 | 测量标准 |
|------|------|----------|
| What-if模拟 | <30秒 | 端到端 |
| Trace查询 | <3秒 | P99 |
| API响应 | <500ms | P95 |
| 页面加载 | <2秒 | 首屏 |
| 并发用户 | 500+ | 同时在线 |

---

## 2. 可用性要求

| 指标 | 要求 | 恢复时间 |
|------|------|----------|
| 系统可用性 | 99.9% | - |
| 计划内维护 | 每月1次 | 2小时内 |
| 故障恢复 | RTO<4h | RPO<1h |

---

## 3. 安全要求

| 控制域 | 控制项 | 要求 |
|--------|--------|------|
| 身份认证 | MFA多因素 | 必须启用 |
| 访问控制 | RBAC+ABAC | 最小权限 |
| 数据加密 | 传输+存储 | AES-256 |
| 审计日志 | 全链路记录 | 保留1年 |

---

## 4. 国际化要求

| 维度 | 支持范围 |
|------|----------|
| 语言 | 中文、英文 |
| 时区 | UTC+8、UTC |
| 货币 | CNY、USD |
| 日期格式 | YYYY-MM-DD、DD/MM/YYYY |
