## 2. 移动端交互规范

### 2.1 ASN 确认流程

```mermaid
flowchart TD
    A[收到PO通知] --> B[查看订单详情]
    B --> C[确认/修改交期]
    C --> D{验证通过?}
    D -->|是| E[提交ASN]
    D -->|否| F[填写原因]
    F --> C
    E --> G[确认成功]
```

## 3. 安全规范

| 控制点 | 要求 |
|--------|------|
| 登录超时 | 30分钟无操作自动退出 |
| 密码策略 | 8位+大小写+数字 |
| 敏感操作 | 短信验证码确认 |
| 数据传输 | HTTPS + AES加密 |
