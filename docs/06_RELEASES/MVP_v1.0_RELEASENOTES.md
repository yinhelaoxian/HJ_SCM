## 5. 部署说明

### 5.1 环境要求

| 组件 | 版本 |
|------|------|
| Node.js | 20+ |
| Java | 17+ |
| PostgreSQL | 14+ |
| Redis | 6+ |

### 5.2 启动命令

```bash
# 前端
cd frontend
npm install
npm run dev

# 后端
cd backend
mvn spring-boot:run
```
