## 菜单结构说明

系统采用 SCOR 五层结构组织菜单：

| 分组 | 功能入口 |
|------|----------|
| **总览** | 供应链指挥中心、智能异常工作台 |
| **计划 · PLAN** | S&OP 产销协同、AI 需求预测、供需平衡工作台、MPS 主生产计划、情景模拟工作台 |
| **采购 · SOURCE** | AI 采购建议、供应商管理 |
| **库存 · STOCK** | 库存工作台、ATP 可承诺量 |
| **交付 · DELIVER** | 订单全链路追踪、预警规则配置 |
| **绩效 · ENABLE** | SCOR 绩效看板、报表中心、决策支持系统 |

## 典型演示路径

### 路径1：Bühler断供危机（推荐）
```
1. 左侧菜单 → 总览 → 智能异常工作台
2. 查看 Bühler 延期预警
3. 点击查看详情 → AI建议
4. 跳转至 采购 · SOURCE → 供应商管理 → 绩效评分管理
5. 查看 Buhler OTD 下滑趋势
6. 跳转至 绩效 · ENABLE → 决策支持系统 → 批准备选供应商认证
```

### 路径2：圣诞旺季备货
```
1. 左侧菜单 → 计划 · PLAN → AI 需求预测
2. 查看圣诞旺季预测
3. 跳转至 库存 · STOCK → ATP 可承诺量
4. 检查 HJ-LA23 ATP 是否充足
5. 跳转至 MPS 主生产计划 → 13周滚动计划
6. 调整生产计划
```

### 路径3：产销协同评审
```
1. 左侧菜单 → 计划 · PLAN → S&OP 产销协同 → 产销平衡总览
2. 查看供需差距
3. 点击 需求版本管理 → 查看版本对比
4. 点击 会议管理 → 查看会议日历和决议
5. 点击 差异分析报告
```

### 路径4：泰国产能爬坡
```
1. 左侧菜单 → 计划 · PLAN → MPS 主生产计划 → RCCP 粗能力核查
2. 查看青岛 A 线超负荷（112%）
3. 查看泰国线利用率不足（43%）
4. 查看 AI 建议：转产泰国可节省 ¥95万
5. 点击"应用此方案"
```

## Sprint 1 演示路径

### 1.1 仪表盘
```
1. 访问 http://localhost:3000/

```

```

```
1. 左侧菜单 → 总览 → 智能异常工作台
